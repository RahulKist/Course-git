Because the basics are important!

This is a crash course on Git and GitHub. It covers the basics of version control, how to use Git, and how to collaborate with others using GitHub.

What is Git?

Git is a distributed version control system that allows you to track changes to your code and collaborate with others. It was created by Linus Torvalds

in 2005 to help manage the development of the Linux kernel. Since then, it has become one of the most popular version control systems in the world.
